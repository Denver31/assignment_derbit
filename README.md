# Crypto Price Collector (Deribit)

Тестовое задание (пример кода) на позицию Backend Developer.

Сервис предназначен для периодического получения цен криптовалют с биржи Deribit, их сохранения в базу данных и предоставления данных через HTTP API.

---

## Функциональность

- Сбор index price для валют каждую минуту:
  - `btc_usd`
  - `eth_usd`
- Сохранение данных в PostgreSQL:
  - тикер
  - цена
  - UNIX timestamp
- HTTP API для получения сохранённых данных
- Фоновая обработка через Celery
- Контейнеризация с помощью Docker Compose
---
## Развёртывание проекта

Проект разворачивается с помощью Docker Compose.

1. Создать файл `.env` на основе примера.
2. Запустить проект командой:
   docker compose up --build
3. При первом запуске автоматически создаются база данных и таблицы.

После запуска API доступно по адресу http://localhost:8000,
документация — http://localhost:8000/docs.
